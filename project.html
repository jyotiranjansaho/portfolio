Employee Management System

The main objective of an employee management system to retrieve data from a large database and display it according to user requirement use three tier architecture. To retrieve this data, we use JDBC (java database connectivity) to connect the java and MYSQL database. In the project, we used DML to write queries and create a database using MYSQL. We also wrote all of the business logic using object-oriented programming concepts within the Eclipse IDE
